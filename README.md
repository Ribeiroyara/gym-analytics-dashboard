# Gym Analytics Dashboard

## Overview

Gym Analytics Dashboard is a data analytics web application built with Python and Streamlit.

The dashboard connects directly to a MySQL database and transforms gym business data into visual insights, including member status, payment performance, workout popularity, class bookings, and revenue metrics.

This project was created to demonstrate practical skills in data analysis, database connection, dashboard development, and business reporting.

---

## Features

- MySQL database connection
- Interactive dashboard built with Streamlit
- Member status filter
- Key business metrics
- Revenue analysis
- Payment status analysis
- Workout popularity chart
- Class popularity chart
- Member data table
- Payment data table
- Members workouts data table
